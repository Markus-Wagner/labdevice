Detailed Description

ProLogix GPIB-ETHERNET (GPIB-LAN) Controller.

Usage: Initalise subclass

pr = ProLogix("eth",% mode
"192.168.73.42",% IP Address
"1234"); % Port
dev = labDevice(deviceName,"eth2gpib",pr,GPIBAddr)
dev.write("hello world");
