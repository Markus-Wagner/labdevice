Detailed Description

Data acquisition unit with some custom hardware added to create a precice rotatable table.

Usage: Initalise subclass

stepper = UASAugsburg_Stepper("gpib",% mode
6); % GPIB address
stepper.nullpos(); %point the antenna in a certain direction
stepper.turn(360); %turn the table by 360 degrees
