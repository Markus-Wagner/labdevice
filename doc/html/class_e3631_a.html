Detailed Description

Keysight E3631A Power Supply.

Usage: Initalise subclass

psu = E3631A("gpib",% mode
6); % GPIB address
psu.setVolts(3,'P6V');
psu.setAmps(1,'P6V');
psu.setOutput('N25V');
psu.setAmps(0.05);
i = psu.getAmpsMeasured('P6V')
